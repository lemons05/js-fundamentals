# js-fundamentals
90-day journey from JS basics to full-stack engineer

## Goal
Going from intermediate HTML/CSS + beginner JavaScritp to a hireable full-stack software engineer (React + Node) by following a structured 90-day plan, then building toward a job-ready portfolio by early 2027.

## Plan
- **Days 1–30:** JavaScript fundamentals — variables, functions, closures,
  async/await, DOM manipulation
- **Days 31–60:** React + Node/Express, connecting frontend to a real backend
- **Days 61–90:** Testing, deployment, open-source contributions, and
  building a job-application-ready portfolio

## Why this repo exists
This is where I'm logging daily practice, small exercises, and notes as I
work through JS fundamentals. Bigger projects will live in their own repos
and get linked here as I build them.
